Building out a VM as a Portainer containerization solution.

### Quick links
* [.. up dir](../../README.md)

## Overview
Storing the VMs at `/var/lib/vms` with a directory per VM.

## Setup

1. Build vm
   ```
   $ ./clu build vm prod1
   ```
2. Run vm
   ```
   $ ./clu run vm prod1
   ```
